# HACP 协议设计

> Human-AI Collaboration Protocol — 人工智能协作规范

## 核心原则

**选择可概率，传递必确定。**

系统在"选择行为"上接受概率性，但在"信息传递"上保持绝对确定性。

## 四个组件

### 1. 追问机制（已验证）

不确定时停下来问用户，模型暴露假设，规则校验结构。

- 模型生成认知显化：理解复述 + 假设清单（标注来源）+ 盲区清单（标注是否阻断）
- 规则引擎扫描文本模式，强制触发追问
- 意图识别：模型语义判断替代关键词匹配

### 2. 上下文编排器（协议设计）

四层注意力位阶：系统提示词区 / 前置区 / 中置区 / 近因区

不同位阶有不同的注意力权重和压缩规则。

### 3. 记忆域数据库（协议设计）

热/温/冷三层记忆系统，标签精确匹配，梦境整合自动沉淀。

### 4. 概率确定（部分验证）

- 选择阶段：模型可以概率性地决定选哪条记忆、用哪种检索策略
- 传递阶段：一旦用户确认，原文完整注入，零压缩零失真
